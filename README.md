# Projeto-Final---Data-Science

Crie um ambiente virtual em python com o comando `python -m venv env`.
Ative o ambiente virtual com o comando `source env/bin/activate`.
Instale as bibliotecas necessárias com: `pip install -r requirements.txt`.

# Execução

Rode o pipeline na ordem dos arquivos, pois outros datasets são criados e salvos, e precisa ser na ordem que estão numerados.
